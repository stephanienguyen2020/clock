# Nebula Clock

A creative, abstract clock visualization built with [p5.js](https://p5js.org/).

## Live Demo
[https://stephanienguyen2020.github.io/clock/](https://stephanienguyen2020.github.io/clock/)

## How to Read
- **Outer Ring (Pink):** Seconds. The tip indicates the current second.
- **Middle Ring (Cyan):** Minutes. The tip indicates the current minute.
- **Inner Ring (Gold):** Hours. The tip indicates the current hour.
- **Ticks:** The background ticks represent the 12 hour/minute marks.
- **Center:** A pulsing nebula core indicating the passage of time.
